# Rascunho

**Sketching systems in public.**

A small blog about thinking through systems — one post at a time.

The name *Rascunho* means **draft** in Portuguese.  
This blog treats ideas the same way: rough, evolving, and continuously refined.

## Writing

Posts explore topics like:

- systems thinking
- building software
- first principles
- learning through making

The goal isn’t perfect conclusions —  
it’s documenting the process of figuring things out.

## Deployment

The blog is deployed using **GitHub Pages** and built with simple static files.

No frameworks.  
Just Markdown, HTML, and a bit of JavaScript.
